# Projeto: Cadastro de Clientes

## Objetivo: 

### Criar uma aplicação MVT que permite executar operações CRUD (Create, Read, Update e Delete) em banco de dados, a partir de uma interface estilizada.

## Tecnologias Utilizadas:

### - Python;

### - Django;

### - HTML;

### - Bootstrap;

### - JQuery;

### - MySQL;
